# my-mood-app
In this app, we were able to follow mood swings during the day, with the support of graphs and extra notes.

## Requirements
- iOS 16+
- Xcode 14
- Swift 5

## This project uses:
- UIKit
- VIPER
- Swift package manager
- Dependency Injection
- Charts
- Lottie
